CMPUT 201: Practical Programming Methodology (Winter 2015)

Lab 5: More Pointers and Arrays

Lab Exercise 5 : More Pointers and Arrays

This lab exercise contributes to your lab mark. Note that this lab exercise will be submitted via astep, NOT demoed. See the bottom of this page for instructions on submitting your lab exercise.

Objectives

The purpose of this lab exercise is to deepen your understanding of pointers and arrays in C. You will also practice reading C code and writing about technical concepts in your short answers to the questions.

Recommended Preparation (in advance of lab)

Review pointers and arrays in C. In King, this is Chapters 11, 12, 13, 14.4, 17.1-17.4. You can also search online for resources on pointers and arrays in C.

Lab Exercise

Please ask your TA if you need help with this Exercise.

Download ex5.tar.gz (see the Lab 5 Resources page) , and extract its contents. It contains a directory Ex5 with the following files:

eg.c
Makefile
questions.txt

  1. In the Ex5 directory, type make eg1, then type make eg2. This makes two executables, eg1 and eg2.
  2. Run eg1, then eg2.
  3. Examine the code in eg.c, and the output from eg1 and eg2, and use this to answer the questions in questions.txt.
    • Write your answers directly in questions.txt as indicated.
    • Do not put your name or any other identification in this file.
    • Answer the questions clearly and concisely.

Submission commands and due dates

Submit questions.txt (which you will have modified).

Submit that file by

  1. going to the directory with questions.txt using cd,
  2. then type in the proper astep command for your lab, listed in the table below.

    Type 'y' when prompted.

LATE LAB SUBMISSIONS WILL NOT BE ACCEPTED.
