CMPUT 201: Practical Programming Methodology (Winter 2015)

Assignment 1b - Sample Input 3

Suppose your inputfile.3 file looked like:
# Comments look like this
Image A 100 100 =
lineTo 50 -100
lineTo 50 100
lineTo -110 -66
lineTo 120 0
lineTo -110 66
End Image A
# ############################
print A
draw A
child pause 5
child clearScreen
translate A 50 50
draw A
child pause 5
child clearScreen
translate A 50 50
draw A
child pause 60
# child end below is last line in input file
child end
And you ran it as such:
 ./draw1b inputfile.3
Your program should produce the following output to stdout:
./draw1b started on Mon 6 Feb 2012 23:21:45 MST
Input file: inputfile.3
# Comments look like this
# ############################
Print Image A = 
100 100 
150 0 
200 100 
90 34 
210 34 
100 100 
End Image A 
# child end below is last line in input file
